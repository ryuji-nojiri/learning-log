# 📚 Learning Log

日々の学習で得た知識や、技術的な備忘録を蓄積していくリポジトリです。
「昨日の自分より1%賢くなる」を目標に更新しています。

## 🛠 現在の注力テーマ
* **Next.js / React**: モダンなフロントエンド開発の習得
* **TypeScript**: 型安全なコーディングの習慣化
* **Git/GitHub**: スムーズな共同開発・管理フローの理解
* **AI**: AI駆動開発や、バイブコーディングなど、AIに関することなら幅広く

## 📂 ディレクトリ構成
年月ごとにフォルダを分けて管理しています。

## 🚀 運用ルール
1. 小さな気づきでも、まずは `git push` する。
2. 完璧な文章を目指さず、未来の自分が理解できるメモを残す。
3. 解決したエラー（ECONNRESETなど）は、原因と対策をセットで記録する。
