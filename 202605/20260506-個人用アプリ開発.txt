# 今日実施したこと
・個人用アプリ開発

１．概要
「今日、どこ行こう？」という迷いを解消し、日常のお出かけのマンネリ化を防ぐアプリです。
Google Maps API を使用し、複数のスポットを巡る「お出かけプラン」と、各場所での「ミッション」をセットで提案します。

２．コア機能（MVP）
モバイルファースト・デザイン:
　・片手で操作しやすいUI設計。
条件指定コースガチャ:
　・都道府県、メインの目的（のんびり、アクティブ、映え、癒やし等）を指定すると、3つのスポットを組み合わせたコースを提案。
移動最適化アルゴリズム:
　・1箇所目のスポットを基準に、近隣エリア（同じ市区町村、または一定半径内）から後続のスポットを選出し、移動コストを最小化。
タイムライン形式のプラン提示:
　Spot 1: 午前（体験・観光）
　Spot 2: ランチ（グルメ）
　Spot 3: 午後（散策・リラックス）

３．技術スタック
Frontend：React（Next.js）
Backend：Next.js（App Router）
Database：Supabase
API：Google Maps API（Google Places API）
Styling：Tailwind CSS / React Icons
PaaS：vercel
